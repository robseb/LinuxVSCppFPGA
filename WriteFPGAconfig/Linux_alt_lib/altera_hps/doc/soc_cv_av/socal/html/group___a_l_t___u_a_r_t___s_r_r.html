Register : Software Reset Register - srr
Component : UART Module - ALT_UART

Description

Provides Software Resets for Tx/Rx FIFO's and the uart.

Field : Rx FIFO Reset - rfr

This is a shadow register for the Rx FIFO Reset bit (FCR[1]). This can be used to remove the burden on software having to store previously written FCR values (which are pretty static) just to reset the receive FIFO. This resets the control portion of the receive FIFO and treats the FIFO as empty. This will also de-assert the DMA Rx request and single signals. Note that this bit is 'self- clearing' and it is not necessary to clear this bit.
