Register : qoshighpri Register - qoshighpri
Register Group : SDRAM Controller Module - ALT_SDR_CTL

Description

Register Layout

Bits Access Reset Description
[19:0] RW Unknown High Priority QoS Value
[31:20] ??? 0x0 UNDEFINED

Field : High Priority QoS Value - highpriorityval

This 20 bit field is a 2 bit field for each of the 10 ports. The field used for each port in this register controls the priority used for a port
