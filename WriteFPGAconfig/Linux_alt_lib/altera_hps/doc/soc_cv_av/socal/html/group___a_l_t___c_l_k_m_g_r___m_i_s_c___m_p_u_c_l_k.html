Register : Main PLL C0 Control Register for Clock mpu_clk - mpuclk
Register Group : Miscellaneous settings registers Group - ALT_CLKMGR_MISCGRP

Description

Contains settings that control clock mpu_clk generated from the C0 output of the Main PLL.

Only reset by a cold reset.

Register Layout

Bits Access Reset Description
[8:0] RW 0x1 Counter
[31:9] ??? 0x0 UNDEFINED

Field : Counter - cnt

Divides the VCO frequency by the value+1 in this field.
