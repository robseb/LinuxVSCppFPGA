Register : FIFO Triple Byte Register - fifotriplebyte
Component : Scan Manager Module Registers - ALT_SCANMGR

Description

Writes to the FIFO Triple Byte Register write a triple byte value to the command FIFO. If the command FIFO is full, the APB write operation is stalled until the command FIFO is not full.

Reads from the Triple Byte FIFO Register read a triple byte value from the command FIFO. If the command FIFO is empty, the APB read operation is stalled until the command FIFO is not empty.

See the ARM documentation for a description of the read and write values.

The name of this register in ARM documentation is BWFIFO3 for writes and BRFIFO3 for reads.
