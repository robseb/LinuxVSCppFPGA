Register : Synchronization Level Register - gpio_ls_sync
Register Group : Configuration Monitor (MON) Registers - ALT_MON

Description

The Synchronization level register is used to synchronize inputs to the l4_mp_clk. All MON interrupts are already synchronized before the GPIO instance so it is not necessary to setup this register to enable synchronization.

Register Layout

Bits Access Reset Description
[0] RW 0x0 Synchronization Level Field
[31:1] ??? 0x0 UNDEFINED

Field : Synchronization Level Field - gpio_ls_sync

The level-sensitive interrupts is synchronized to l4_mp_clk.
