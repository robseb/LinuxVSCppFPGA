Register : Raw Interrupt Status Register - gpio_raw_intstatus
Component : GPIO Module - ALT_GPIO

Description

This is the Raw Interrupt Status Register for Port A Data Register. It is used with the Interrupt Mask Register to allow interrupts from the Port A Data Register.

Register Layout

Bits Access Reset Description
[28:0] RW 0x0 Raw Interrupt Status Field
[31:29] ??? 0x0 UNDEFINED

Field : Raw Interrupt Status Field - gpio_raw_intstatus
