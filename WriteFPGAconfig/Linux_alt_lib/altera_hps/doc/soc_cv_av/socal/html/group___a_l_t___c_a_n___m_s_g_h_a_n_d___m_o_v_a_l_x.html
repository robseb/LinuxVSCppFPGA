Register : Message Valid X Register - MOVALX
Register Group : Message Handler Group - ALT_CAN_MSGHAND

Description

Reading this register allows the CPU to quickly detect if any of the message valid bits in each of the MOVALA, MOVALB, MOVALC, and MOVALD Message Valid Registers are set.

Register Layout

Bits Access Reset Description
[0] R 0x0 MsgValA
[1] R 0x0 MsgValA
[2] R 0x0 MsgValA
[3] R 0x0 MsgValA
[4] R 0x0 MsgValB
[5] R 0x0 MsgValB
[6] R 0x0 MsgValB
[7] R 0x0 MsgValB
[8] R 0x0 MsgValC
[9] R 0x0 MsgValC
[10] R 0x0 MsgValC
[11] R 0x0 MsgValC
[12] R 0x0 MsgValD
[13] R 0x0 MsgValD

[15] R 0x0 MsgValD
[31:16] ??? 0x0 UNDEFINED
