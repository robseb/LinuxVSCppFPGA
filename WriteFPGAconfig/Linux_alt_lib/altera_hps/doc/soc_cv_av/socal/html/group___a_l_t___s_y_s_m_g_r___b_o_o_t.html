Register : Boot Info Register - bootinfo
Component : System Manager Module - ALT_SYSMGR

Description

Provides access to boot configuration information.

Register Layout

Bits Access Reset Description

[4:3] R Unknown Clock Select

[9:8] R Unknown HPS Pin Clock Select

Field : Boot Select - bsel

The boot select field specifies the boot source. It is read by the Boot ROM code on a cold or warm reset to determine the boot source.

The HPS BSEL pins value are sampled upon deassertion of cold reset.

Field : Clock Select - csel

The clock select field specifies clock information for booting. The clock select encoding is a function of the CSEL value. The clock select field is read by the Boot ROM code on a cold or warm reset when booting from a flash device to get information about how to setup the HPS clocking to boot from the specified clock device.

The encoding of the clock select field is specified by the enum associated with this field.

The HPS CSEL pins value are sampled upon deassertion of cold reset.
