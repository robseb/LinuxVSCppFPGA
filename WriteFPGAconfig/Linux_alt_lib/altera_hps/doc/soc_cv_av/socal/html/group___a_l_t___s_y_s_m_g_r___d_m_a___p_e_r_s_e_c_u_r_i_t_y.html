Register : Peripheral Security Register - persecurity
Register Group : DMA Controller Group - ALT_SYSMGR_DMA

Description

Controls the security state of a peripheral request interface. Sampled by the DMA controller when it exits from reset.

These register bits should be updated during system initialization prior to removing the DMA controller from reset. They may not be changed dynamically during DMA operation.

Register Layout

Bits Access Reset Description
[31:0] RW 0x0 Peripheral Non-Secure

Field : Peripheral Non-Secure - nonsecure

If bit index [x] is 0, the DMA controller assigns peripheral request interface x to the Secure state.

If bit index [x] is 1, the DMA controller assigns peripheral request interface x to the Non-secure state.

Reset by a cold or warm reset.
