Register : ECC Single Bit Error Count Register - sbecount

Register Layout

Bits Access Reset Description
[7:0] RW Unknown Single Bit Error Count
[31:8] ??? 0x0 UNDEFINED

Field : Single Bit Error Count - count

Reports the number of single bit errors that have occurred since the status register counters were last cleared.
