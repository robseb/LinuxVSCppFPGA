Field : Mask Dropped Auto-correction Interrupt - corrdropmask

Set this bit to a one to mask interrupts for an ECC correction write back needing to be dropped. This indicates a burst of memory errors in a short period of time.

Field : Clear Interrupt Signal - intrclr

Writing to this self-clearing bit clears the interrupt signal. Writing to this bit also clears the error count and error address registers.
