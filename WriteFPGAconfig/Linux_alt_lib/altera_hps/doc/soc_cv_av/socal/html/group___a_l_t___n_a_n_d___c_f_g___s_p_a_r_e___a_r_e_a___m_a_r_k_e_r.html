Register : spare_area_marker
Register Group : Configuration registers - ALT_NAND_CFG

Description

Spare area marker value

Register Layout

Bits Access Reset Description
[15:0] RW 0xffff ALT_NAND_CFG_SPARE_AREA_MARKER_VALUE
[31:16] ??? 0x0 UNDEFINED

Field : value

The value that will be written in the spare area skip bytes. This value will be used by controller while in the MAIN mode of data transfer. Only the least- significant 8 bits of the field value are used.
