Register : Transmit FIFO Read Register - tfr
Component : UART Module - ALT_UART

Description

Used in FIFO Access test mode.

Register Layout

Bits Access Reset Description
[7:0] R 0x0 Transmit FIFO Read
[31:8] ??? 0x0 UNDEFINED

Field : Transmit FIFO Read - tfr

These bits are only valid when FIFO access mode is enabled (FAR[0] is set to one). When FIFO's are enabled, reading this register gives the data at the top of the transmit FIFO. Each consecutive read pops the transmit FIFO and gives the next data value that is currently at the top of the FIFO. When FIFO's are not enabled, reading this register gives the data in the THR.
