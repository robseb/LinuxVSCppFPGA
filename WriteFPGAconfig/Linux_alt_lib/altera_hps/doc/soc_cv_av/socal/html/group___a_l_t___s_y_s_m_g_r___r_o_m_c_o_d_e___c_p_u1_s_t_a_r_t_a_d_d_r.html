Register : CPU1 Start Address Register - cpu1startaddr
Register Group : Boot ROM Code Register Group - ALT_SYSMGR_ROMCODE

Description

When CPU1 is released from reset and the Boot ROM is located at the CPU1 reset exception address (the typical case), the Boot ROM reset handler code reads the address stored in this register and jumps it to hand off execution to user software.

Register Layout

Bits Access Reset Description
[31:0] RW 0x0 Address

Field : Address - value

Address for CPU1 to start executing at after coming out of reset.
