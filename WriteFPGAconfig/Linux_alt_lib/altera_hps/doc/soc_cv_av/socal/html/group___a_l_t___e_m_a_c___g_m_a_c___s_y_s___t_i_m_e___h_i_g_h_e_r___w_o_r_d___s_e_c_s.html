Register : Register 457 (System Time - Higher Word Seconds Register) - System_Time_Higher_Word_Seconds
Register Group : GMAC Register Group - ALT_EMAC_GMAC

Description

System time higher word

Register Layout

Bits Access Reset Description
[15:0] RW 0x0 Timestamp Higher Word Register
[31:16] ??? 0x0 UNDEFINED

Field : Timestamp Higher Word Register - tshwr

This field contains the most significant 16-bits of the timestamp seconds value. The register is directly written to initialize the value. This register is incremented when there is an overflow from the 32-bits of the System Time - Seconds register.
