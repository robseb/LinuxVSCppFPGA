Register : Device IN Endpoint FIFO Empty Interrupt Mask Register - diepempmsk
Register Group : Device Mode Registers - ALT_USB_DEV

Description

This register is used to control the IN endpoint FIFO empty interrupt generation (DIEPINTn.TxfEmp).
