Register : Control Register - ctrl
Register Group : SDMMC Controller Group - ALT_SYSMGR_SDMMC

Description

Registers used by the SDMMC Controller. All fields are reset by a cold or warm reset.

Field : Feedback Clock Select - fbclksel

Select which fb_clk to be used as cclk_in_sample.

If 0, cclk_in_sample is driven by internal phase shifted cclk_in.

If 1, cclk_in_sample is driven by fb_clk_in. No phase shifting is provided internally on cclk_in_sample.

Note: Using the feedback clock (setting this bit to 1) is not a supported use model.
