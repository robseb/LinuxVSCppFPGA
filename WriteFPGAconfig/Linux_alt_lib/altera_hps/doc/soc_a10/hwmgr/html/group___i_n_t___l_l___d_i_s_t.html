Interrupt Controller Distributor Interface [Secure]
Interrupt Controller Low-Level API [Secure]

Description

This group of APIs provide access, configuration, and control of the Generic Interrupt Controller (GIC) Distributor interface when in the Secure state.

ALT_STATUS_CODE alt_int_dist_secure_enable ( ALT_INT_INTERRUPT_t  int_id)

Enable the specified interrupt for secure operation.

This function configures the interrupt as Secure.

All interrupts are secure by default.

Parameters
int_idThe interrupt identifier. All interrupts are valid, 0 - 1019.
Return values
ALT_E_SUCCESSThe operation was successful.
ALT_E_ERRORThe operation failed.

ALT_STATUS_CODE alt_int_dist_enable ( ALT_INT_INTERRUPT_t  int_id)

Enable the specified interrupt to be forwarded to the CPU interface.

For Software Generated Interrupts (SGI) (interrupts 0 - 15) and Private Peripheral Interrupt (PPI) (interrupts 16 - 32), interrupts must be enabled on each CPU that will service the interrupt. This is done by calling this API when executing on that CPU.

Parameters
int_idThe interrupt identifier. All interrupts are valid, 0 - 1019.
Return values
ALT_E_SUCCESSThe operation was successful.
ALT_E_ERRORThe operation failed.
ALT_E_BAD_ARGThe given interrupt identifier is invalid.
