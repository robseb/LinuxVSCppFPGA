Register : Initial Software State Register - romcode_initswstate
Component : ALT_SYSMGR_ROM

Description

Initial Software loaded by the Boot ROM writes the magic value 0x49535756 (ISWV in ASCII) to this register when it has reached a valid state.

Reset only on a cold reset.

Register Layout

Bits Access Reset Description
[31:0] RW 0x0 Value

Field : Value - value

Written with magic value.
