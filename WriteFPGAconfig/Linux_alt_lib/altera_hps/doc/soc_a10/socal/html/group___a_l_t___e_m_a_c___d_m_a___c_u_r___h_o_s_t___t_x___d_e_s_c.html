Description

Register 18 (Current Host Transmit Descriptor Register)

The Current Host Transmit Descriptor register points to the start address of the current Transmit Descriptor read by the DMA.

Register Layout

Bits Access Reset Description
[31:0] R 0x0 ALT_EMAC_DMA_CUR_HOST_TX_DESC_CURTDESAPTR

Field : curtdesaptr

Host Transmit Descriptor Address Pointer

Cleared on Reset. Pointer updated by the DMA during operation.
