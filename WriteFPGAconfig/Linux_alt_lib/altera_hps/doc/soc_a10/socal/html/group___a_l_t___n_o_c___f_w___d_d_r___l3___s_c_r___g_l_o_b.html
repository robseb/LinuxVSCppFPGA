Register : global
Component : ALT_NOC_FW_DDR_L3_SCR

Description

Global Firewall Control Register. This register will store various overrides that change default firewall behavior on the entire interconnect.

Register Layout

Bits Access Reset Description
[0] RW 0x0 ALT_NOC_FW_DDR_L3_SCR_GLOB_ERROR_RESPONSE
[31:1] ??? 0x0 UNDEFINED

Field : error_response

When 0, transactions blocked by the firewall will silently fail (i.e successful completion with random data). When 1, transactions blocked by the firewall will return error
