Component : ALT_EMAC

Description

Register 87 (Transmit Frame Count for Excessive Collision Error Frames)

This register maintains the number of frames aborted because of excessive (16) collision error.

Register Layout

Bits Access Reset Description
[31:0] R 0x0 ALT_EMAC_GMAC_TXEXESSCOL_CNT

Field : cnt

This field indicates the number of frames aborted because of excessive (16) collision error.
