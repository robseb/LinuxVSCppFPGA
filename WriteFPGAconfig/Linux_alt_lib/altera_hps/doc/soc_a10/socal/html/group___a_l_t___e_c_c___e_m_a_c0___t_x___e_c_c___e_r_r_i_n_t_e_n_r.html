Register : ERRINTENR
Component : ALT_ECC_EMAC0_TX_ECC

Description

Error Interrupt reset.

Register Layout

Bits Access Reset Description
[0] RW 0x0 ALT_ECC_EMAC0_TX_ECC_ERRINTENR_SERRINTR
[31:1] ??? 0x0 UNDEFINED

Field : SERRINTR

This bit is used to reset the single-bit error interrupt bit. o

Reads reflect SERRINTEN.

1'b0: Writing of zero has no effect.

1'b1: By writing one, this bit will reset SERRINTEN bit to 0. This is performing a bitwise writing of this feature.
