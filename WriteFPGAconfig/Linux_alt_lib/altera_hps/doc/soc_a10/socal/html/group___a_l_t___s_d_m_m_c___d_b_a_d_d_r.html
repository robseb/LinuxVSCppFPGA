Register : dbaddr
Component : ALT_SDMMC

Description

Descriptor List Base Address Register

Register Layout

Bits Access Reset Description
[31:0] RW 0x0 ALT_SDMMC_DBADDR_SDL

Field : sdl

Start of Descriptor List. Contains the base address of the First Descriptor.

The LSB bits [0/1/2:0] for 16/32/64-bit bus-width) are ignored and taken as all- zero by the IDMAC internally. Hence these LSB bits are read-only.
