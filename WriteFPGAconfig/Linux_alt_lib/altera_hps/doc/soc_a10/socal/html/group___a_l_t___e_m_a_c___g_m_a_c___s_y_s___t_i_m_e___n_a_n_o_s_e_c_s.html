Register : Register 451 (System Time - Nanoseconds Register) - gmacgrp_system_time_nanoseconds
Component : ALT_EMAC

Description

The value in this field has the sub second representation of time, with an accuracy of 0.46 ns. When TSCTRLSSR is set, each bit represents 1 ns and the maximum value is 0x3B9A_C9FF, after which it rolls-over to zero.

Register Layout

Bits Access Reset Description
[30:0] R 0x0 Timestamp Sub Seconds
[31] ??? 0x0 UNDEFINED

Field : Timestamp Sub Seconds - tsss

The value in this field has the sub second representation of time, with an accuracy of 0.46 ns. When bit 9 (TSCTRLSSR) is set in Register 448 (Timestamp Control Register), each bit represents 1 ns and the maximum value is 0x3B9A_C9FF, after which it rolls-over to zero.
