Field : Address Transfer Type for Standard SPI modes - addrwidth

0 : Addresses can be shifted to the device on DQ0 only 1 : Addresses can be shifted to the device on DQ0 and DQ1 only 2 : Addresses can be shifted to the device on DQ0, DQ1, DQ2 and DQ3

Field Enumeration Values:

Enum Value Description
ALT_QSPI_DEVWR_ADDRWIDTH_E_SINGLE 0x0 Write address transferred on DQ0. Supported by
: all SPI flash devices
ALT_QSPI_DEVWR_ADDRWIDTH_E_DUAL 0x1 Read address transferred on DQ0 and DQ1.
: Supported by some SPI flash devices that support
: the Extended SPI Protocol and by all SPI flash
: devices that support the Dual SP (DIO-SPI)
: Protocol.
ALT_QSPI_DEVWR_ADDRWIDTH_E_QUAD 0x2 Read address transferred on DQ0, DQ1, DQ2, and
: DQ3. Supported by some SPI flash devices that
: support the Extended SPI Protocol and by all SPI
: flash devices that support the Quad SP (QIO-SPI)
: Protocol.

Field : Data Transfer Type for Standard SPI modes - datawidth

0 : SIO mode data is shifted to the device on DQ0 only and from the device on DQ1 only 1 : Used for Dual Input/Output instructions. For data transfers, DQ0 and DQ1 are used as both inputs and outputs. 2 : Used for Quad Input/Output instructions. For data transfers, DQ0,DQ1,DQ2 and DQ3 are used as both inputs and outputs.

Enum Value Description
