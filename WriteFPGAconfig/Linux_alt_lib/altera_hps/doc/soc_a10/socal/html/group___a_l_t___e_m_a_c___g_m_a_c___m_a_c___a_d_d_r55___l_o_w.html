Description

Register 591 (MAC Address55 Low Register)

The MAC Address55 Low register holds the lower 32 bits of the 56th 6-byte MAC address of the station.

Register Layout

Bits Access Reset Description
[31:0] RW 0xffffffff ALT_EMAC_GMAC_MAC_ADDR55_LOW_ADDRLO

Field : addrlo

MAC Address55 [31:0]

This field contains the lower 32 bits of the 56th 6-byte MAC address. The content of this field is undefined until loaded by the Application after the initialization process.
