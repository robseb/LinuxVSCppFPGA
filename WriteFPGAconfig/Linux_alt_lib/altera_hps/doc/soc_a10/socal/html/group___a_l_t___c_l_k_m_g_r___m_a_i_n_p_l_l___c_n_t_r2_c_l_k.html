Register : Main PLL Control Register for Counter 2 Clock - cntr2clk
Component : ALT_CLKMGR_MAINPLL

Description

Contains settings that control Couner 2 clock generated from the Main PLL VCO clock.

Register Layout

Bits Access Reset Description
[10:0] RW 0x0 Counter
[31:11] ??? 0x0 UNDEFINED

Field : Counter - cnt

Divides the VCO frequency by the value+1 in this field.
