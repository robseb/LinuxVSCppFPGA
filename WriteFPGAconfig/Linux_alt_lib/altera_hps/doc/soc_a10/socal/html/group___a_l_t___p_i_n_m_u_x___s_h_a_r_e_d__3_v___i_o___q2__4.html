Register : Shared IO 48 Q2 4 Mux Selection Register - pinmux_shared_io_q2_4
Component : ALT_PINMUX_SHARED_3V_IO_GRP

Description

This register is used to control the peripherals connected to shared IO48 pin Q2 4

Only reset by a cold reset (ignores warm reset).

NOTE: These registers should not be modified after IO configuration.There is no support for dynamically changing the Pin Mux selections.

Register Layout

[31:4] R 0x0 ALT_PINMUX_SHARED_3V_IO_Q2_4_RSVD
