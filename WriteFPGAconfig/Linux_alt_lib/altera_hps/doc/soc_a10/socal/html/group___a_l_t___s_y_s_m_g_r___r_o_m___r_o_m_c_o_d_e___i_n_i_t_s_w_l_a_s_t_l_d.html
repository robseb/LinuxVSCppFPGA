Register : Initial Software Last Image Loaded Register - romcode_initswlastld
Component : ALT_SYSMGR_ROM

Description

Contains the index of the last Initial Software image loaded by the Boot ROM from the boot device.

Reset only on a cold reset.

Register Layout

Bits Access Reset Description
[1:0] RW 0x0 Index
[31:2] ??? 0x0 UNDEFINED

Field : Index - index

Index of last image loaded.
