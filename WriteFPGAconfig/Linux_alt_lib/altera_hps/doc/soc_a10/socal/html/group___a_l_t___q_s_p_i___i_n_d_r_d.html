Register : Indirect Read Transfer Control Register - indrd
Component : ALT_QSPI

Description

Register Layout

Bits Access Reset Description
[0] W 0x0 Start Indirect Read
[1] W 0x0 Cancel Indirect Read
[2] R 0x0 Indirect Read Status
[3] RW 0x0 SRAM Full
[4] R 0x0 Queued Indirect Read Operations
[5] RW 0x0 Indirect Completion Status
[7:6] R 0x0 Completed Indirect Operations
[31:8] R 0x0 Reserved

Field : Start Indirect Read - start

Writing a 1 to this bit will trigger an indirect read operation. The assumption is that the indirect start address and the indirect number of bytes register is setup before triggering the indirect read operation.

Field Enumeration Values:

Enum Value Description
ALT_QSPI_INDRD_START_E_DISD 0x0 No Indirect Read
ALT_QSPI_INDRD_START_E_END 0x1 Trigger Indirect Read

Field : Completed Indirect Operations - num_ind_ops_done

This field contains the number of indirect operations which have been completed. This is used in conjunction with the indirect completion status field (bit 5). It is incremented by hardware when an indirect operation has completed. Write a 1 to bit 5 of this register to decrement it.
