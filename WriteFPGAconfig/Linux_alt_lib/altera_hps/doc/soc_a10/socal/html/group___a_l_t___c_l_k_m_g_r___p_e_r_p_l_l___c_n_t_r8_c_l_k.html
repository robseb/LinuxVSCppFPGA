Register : Peripheral PLL Control Register for Counter 8 Clock - cntr8clk
Component : ALT_CLKMGR_PERPLL

Description

Contains settings that control Couner 8 clock generated from the Peripheral PLL VCO clock.

Register Layout

Bits Access Reset Description
[10:0] RW 0x0 Counter
[15:11] ??? 0x0 UNDEFINED
[18:16] RW 0x0 ALT_CLKMGR_PERPLL_CNTR8CLK_SRC
[31:19] ??? 0x0 UNDEFINED

Field : Counter - cnt

Divides the VCO frequency by the value+1 in this field.

Selects the source for the active 5:1 clock selection when the PLL is not bypassed.
