Register : gmacgrp_version
Component : ALT_EMAC

Description

Register 8 (Version Register)

The Version registers identifies the version of the DWC_gmac. This register contains two bytes: one that Synopsys uses to identify the core release number, and the other that you set during core configuration.

Register Layout

Bits Access Reset Description
[7:0] R 0x37 ALT_EMAC_GMAC_VER_SNPSVER
[15:8] R 0x10 ALT_EMAC_GMAC_VER_USERVER
[31:16] R 0x0 ALT_EMAC_GMAC_VER_RSVD_31_16
