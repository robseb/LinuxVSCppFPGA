Register : gmacgrp_rxlengtherror
Component : ALT_EMAC

Description

Register 114 (Receive Frame Count for Length Error Frames)

This register maintains the number of frames received with length error (Length type field not equal to frame size) for all frames with valid length field.

Register Layout

Bits Access Reset Description
[31:0] R 0x0 ALT_EMAC_GMAC_RXLENERROR_CNT

Field : cnt

This field indicates the number of frames received with length error (Length type field not equal to frame size) for all frames with valid length field.
